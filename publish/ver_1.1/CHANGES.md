# ver_1.1 (2026-07-06)

- **다국어 지원 (영어 / 한국어) 추가**
  - 이유: 영어 사용자도 쓸 수 있도록. 문자열을 언어별 JSON(`Locales/en.json`, `ko.json`)으로 분리해 하드코딩 제거. 기본 언어는 영어이며, 설정 → 언어에서 전환.
- **'시작 시 자동 차단' 옵션 제거, 실행 시 항상 차단으로 단순화**
  - 이유: 앱이 켜지면 항상 차단하도록 통일. 차단 해제는 해당 실행 동안만 유효하고 재시작하면 다시 차단.
- **'Windows 시작 시 자동 실행' 기본값 ON**
  - 이유: 처음 설치·실행 시 자동으로 작업 스케줄러에 등록되어, 부팅 → 자동 실행 → 워터마크 차단까지 손 안 대고 진행. 이후 사용자가 직접 끄면 그 선택을 존중.
- **내부 정리**
  - 이유: `DisableBlocking`을 결과 enum 반환으로 리팩토링해 UI 계층에서 다국어 메시지 처리. 설정 저장의 임시 디버그 로그 제거.
