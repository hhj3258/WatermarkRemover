# ver_0.4 (2026-04-09)

- 트레이 메뉴 다크 테마 + 라운드 코너 UI 적용
  - 이유: 기본 WinForms 메뉴가 밋밋해서 모던한 디자인으로 변경
- 상태 표시 배경색 적용 (초록/노랑/주황)
  - 이유: 비활성 회색 텍스트로는 상태가 눈에 띄지 않음
- 차단 해제 후 미재시작 상태 표시 추가 ("⏳ 재시작 후 차단 해제 예정")
  - 이유: 차단 해제 눌렀는데 텍스트가 변하지 않아 혼동 발생
- 토글 분기 로직 버그 수정: IsBlocking() → BlockingEnabled 기준으로 변경
  - 이유: 재시작 대기 상태에서 "다시 차단" 누르면 차단 해제 팝업이 다시 뜨는 버그
