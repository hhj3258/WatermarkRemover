# ver_1.0 (2026-07-06)

정식 버전 1.0 릴리스 — 기능 안정화 후 첫 정식 배포.

- **워터마크 윈도우 실시간 숨김 (SetWinEventHook)**
  - 이유: 풀스크린 게임(리그 오브 레전드 등) 전환 시 워터마크가 깜빡이던 문제 해결.
    폴링만으로는 워터마크 재생성 속도를 따라가지 못해, 윈도우 생성/표시 이벤트를 후킹해 즉시 숨기도록 변경.
- **전면 리팩토링 + 코딩 스타일 스킬 반영**
  - 이유: `WatermarkKiller` → `WatermarkBlocker` 등 실제 동작에 맞는 네이밍으로 통일, `Settings`/`Logger` 분리.
- **트레이 설정 메뉴 완비**
  - 이유: 갱신 주기 / Windows 시작 시 자동 실행 / 시작 시 자동 차단 / 동작 로그 기록 토글 제공.
